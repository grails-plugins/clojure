create-clojure-script

Purpose

The create-clojure-script command creates a Clojure source file under src/main/clj.

Usage

grails create-clojure-script [name]

Example

$ grails create-clojure-script sample
| Rendered template template.clj to destination src/main/clj/sample.clj

That will create a Clojure source file which may be used as a starting point.

(ns grails)
(defn sample_fn []
    (println "sample_fn function called."))
